# Batch Rename Tool PRD (批量重命名工具产品需求文档)

## 1. 产品概述
一款面向 Windows 平台的轻量级、高性能的批量重命名小工具。

## 2. 核心功能需求

### 2.1 目录选择与读取
- 选中文件夹后，极速读取文件夹内所有文件。

### 2.2 表格展示与交互
- **数据列**：序号、原文件名、新文件名 (可编辑)、状态。
- **列级操作按钮 (始终显示)**：
  - 在“原文件名”表头：提供始终显示的一键**复制**按钮。
  - 在“新文件名”表头：提供始终显示的一键**复制**与**粘贴**按钮。点击一键粘贴可直接读取剪贴板的多行文本覆盖到当前列表。
- **操作反馈 (Toast)**：发生复制、粘贴等快捷操作时，提供短暂、美观的屏幕中央悬浮 Toast 提示，告知用户操作结果。
- **单元格粘贴覆盖**：在“新文件名”列，同样支持选中单个输入框按 `Ctrl+V` 粘贴时按顺序覆盖对应的单元格。
- **后缀名编辑开关 (全局联动)**：
  - 提供全局开关，控制是否显示后缀名。
  - **开启**：原文件名与新文件名均完整展示。
  - **关闭**：隐藏扩展名，原文件名和新文件名输入框均**仅显示基础文件名**。所有批量修改、粘贴覆盖也仅作用于基础文件名。

### 2.3 批量规则操作
- 增加前缀、增加后缀、文本替换。
- **重置功能**：提供全局重置按钮，能够将当前所有“新文件名”的修改恢复为初始原文件名状态，并清空所有操作输入框。
