# Batch Rename Tool 技术架构说明

## 1. 技术栈选型
Tauri 2.0 (Rust + Vue 3 + TailwindCSS v4)。

## 2. 核心模块划分

### 2.1 Rust Core
- `list_files` 和 `rename_files` 提供纯粹的文件系统读写能力。

### 2.2 Frontend UI (Vue 3)
- **状态管理**: 增加 `showExtension` 响应式开关。文件对象分为 `baseName`（基础名）和 `ext`（扩展名），绑定到视图的 `editValue`。
- **粘贴处理**: 监听 `paste` 事件拦截换行文本并向下覆盖 `editValue`。
- **复制处理**: 获取 `editValue` 或原始 `name` 的数组，用 `\n` 拼接并写入 `navigator.clipboard`。
